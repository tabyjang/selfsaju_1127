  ⚙️ 현재 구현된 기능

  1. 정확한 사주 계산

  - 생년월일시 → 사주팔자 자동 계산
  - 절기 기반 월주 계산
  - 출생 지역별 시간대 보정 (13개 지역)
  - 야자시 적용 옵션

  2. 사주 명반 표시

  - 4기둥 (년월일시) 천간/지지
  - 십신, 십이운성, 지장간
  - 오행 분포 통계
  - 월령, 천을귀인

  3. 고급 명리 분석

  - 형충회합: 지지 간 상호작용 (충·형·삼합·반합·육합·방합)
  - 신살: 15종 이상 (천을귀인, 도화살, 양인살 등)
  - 대운: 10년 단위 흐름 (순행/역행)
  - 세운: 향후 10년 흐름

  4. AI 심층 분석 (3단계 구조)

  1. 일주/오행 분석 + 핵심 진단 + 개운법
  2. 재물운, 건강운, 애정운
  3. 대운 흐름 + 미래 예측

  5. 대화형 AI 채팅

  - 사주 분석 결과 기반 추가 질문
  - 스트리밍 응답

  6. UI/UX

  - 오행별 색상 코드 (木/火/土/金/水)
  - 유리 효과 카드 디자인
  - 반응형 디자인
  - 토글 가능한 섹션

  🔧 기술 스택
