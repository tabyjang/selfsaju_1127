# 내격(內格) 판단 로직 상세 정리

## 📌 내격이란?
- 일반적인 격국 (약 80~90%의 사주가 내격)
- 월지(月支)를 기준으로 격국을 판단
- 외격(특수격)이 **아닌** 경우 내격으로 판단

---

## 🎯 내격 판단의 핵심 원칙

### 1. 월령(月令)이 주인공
```
격국의 80%는 월지(월령)가 결정합니다.
월지의 지장간(地藏干)을 정확히 알아야 합니다.
```

### 2. 투출(透出)의 중요성
```
지장간의 글자가 천간(天干)에 나타나야 격이 됩니다.
단, 왕지(자오묘유)와 생지(인신사해)는 예외:
- 투출하지 않아도 본기가 격이 될 수 있음
```

### 3. 격의 명칭은 십성(十星) 관계
```
일간(日干) vs 투출된 천간의 관계로 결정
지장간의 음양이 아니라, 천간에 나온 글자의 음양으로 판단!

예시:
- 진토의 지장간: 계수(癸水, 음)
- 천간에 임수(壬水, 양) 투출
- 일간이 병화(丙火, 양)라면?
  → 병화(양) vs 임수(양) → 편관격(칠살격)
  (지장간이 계수라도 상관없음!)
```

## 📊 월지(月支) 분류별 내격 판단 로직

### 🔵 A. 자오묘유(子午卯酉) - 왕지(旺地) ⭐ 가장 단순

#### 특징
- 오행의 기운이 가장 왕성한 곳
- **본기(本氣)만 존재** (지장간이 하나)
- 투출 여부와 관계없이 본기가 그대로 격

#### 지장간 데이터 (사왕지 - 四旺地)
| 월지 | 초기 | 중기 | 본기 | 오행 | 비고 |
|------|------|------|------|------|------|
| 자(子) | 임(壬, 10일) | - | 계(癸, 20일) | 수(水) | 겨울 왕지 (순수한 수 기운) |
| 오(午) | 병(丙, 10일) | 기(己, 9일) | 정(丁, 11일) | 화(火) | 여름 왕지 (기토 섞임) |
| 묘(卯) | 갑(甲, 10일) | - | 을(乙, 20일) | 목(木) | 봄 왕지 (순수한 목 기운) |
| 유(酉) | 경(庚, 10일) | - | 신(辛, 20일) | 금(金) | 가을 왕지 (순수한 금 기운) |

⚠️ **왕지 특징**
- 자, 묘, 유: 중기 없음 (10일 + 20일, 순수한 기운)
- 오(午): 기토(己土) 섞여있음 (10일 + 9일 + 11일)
- 왕지는 본기가 매우 강하므로 투출 여부와 관계없이 본기가 격

#### 로직 (Algorithm)
```javascript
function 왕지_격국_판단(월지, 일간) {
  const 본기 = 왕지_본기[월지]; // 자→계, 오→정, 묘→을, 유→신
  const 십성 = calculate십성(일간, 본기);

  return {
    격용신: 본기,
    격명칭: 십성 + "격",  // 정관격, 편재격 등
    설명: `${월지}월은 ${본기}의 왕지입니다. 본기가 그대로 격이 됩니다.`
  };
}
```

#### 예시
```
사주: 甲 丙 丙 戊
     子 子 午 申

월지: 子(자수) → 본기 계수(癸)
일간: 丙(병화)
관계: 병화 vs 계수 → 정관(음양 다름)
결과: 정관격
```

### 🟢 B. 인신사해(寅申巳亥) - 생지(生地) ⭐ 중간 난이도

#### 특징
- 오행의 기운이 막 생겨나는 곳 (생명력이 강함)
- **여기(餘氣), 중기(中氣), 본기(本氣) 3개 존재**
- 투출 우선순위 존재
- 투출하지 않아도 본기가 격 (생지는 기운이 강함)

#### 지장간 데이터 (사생지 - 四生地)
| 월지 | 월령초기(7일) | 중기(7일) | 본기(16일) | 설명 |
|------|---------------|-----------|------------|------|
| 인(寅) | 무(戊, 토) | 병(丙, 화) | 갑(甲, 목) | 겨울(丑)의 남은 기운(戊) → 장생하는 불(丙) → 본체 나무(甲) |
| 신(申) | 무(戊, 토) | 임(壬, 수) | 경(庚, 금) | 여름(未)의 남은 기운(戊) → 장생하는 물(壬) → 본체 금(庚) |
| 사(巳) | 무(戊, 토) | 경(庚, 금) | 병(丙, 화) | 봄(辰)의 남은 기운(戊) → 장생하는 금(庚) → 본체 불(丙) |
| 해(亥) | 무(戊, 토) | 갑(甲, 목) | 임(壬, 수) | 가을(戌)의 남은 기운(戊) → 장생하는 나무(甲) → 본체 물(壬) |

⚠️ **중요 원칙**
- **여기(餘氣)는 모두 무토(戊土)** - 전달의 남은 기운
- 총 30일: 7일 + 7일 + 16일
- 대부분의 만세력과 이론서는 7-7-16을 정설로 사용

#### 로직 (Algorithm)
```javascript
function 생지_격국_판단(월지, 천간4글자, 일간) {
  const 지장간 = 생지_지장간[월지]; // { 여기, 중기, 본기 }

  // Step 1: 본기가 투출했는가?
  if (천간4글자.includes(지장간.본기)) {
    return {
      격용신: 지장간.본기,
      격명칭: calculate십성(일간, 지장간.본기) + "격"
    };
  }

  // Step 2: 중기가 투출했는가?
  if (천간4글자.includes(지장간.중기)) {
    return {
      격용신: 지장간.중기,
      격명칭: calculate십성(일간, 지장간.중기) + "격"
    };
  }

  // Step 3: 여기(무토)가 투출했는가? (조건부)
  if (천간4글자.includes('戊')) {
    // 사월만 무토 인정 (화토동법)
    if (월지 === '巳') {
      return {
        격용신: '戊',
        격명칭: calculate십성(일간, '戊') + "격",
        특이사항: "사화월 화토동법으로 무토의 힘이 있음"
      };
    }
    // 인월, 신월, 해월: 무토 불인정 (힘이 약함)
    // → 본기로 이동
  }

  // Step 4: 아무것도 투출 안됨
  // → 생지는 기운이 강하므로 본기를 격으로 사용
  return {
    격용신: 지장간.본기,
    격명칭: calculate십성(일간, 지장간.본기) + "격",
    투출여부: false,
    설명: "투출하지 않았으나 생지의 본기가 격이 됨"
  };
}
```

#### 예시 1: 본기 투출
```
사주: 甲 丙 丙 戊
     寅 寅 午 申

월지: 寅(인목) → 본기 갑목(甲)
천간: 甲이 년간에 투출
일간: 丙(병화)
관계: 병화 vs 갑목 → 편인(음양 같음)
결과: 편인격
```

#### 예시 2: 중기 투출
```
사주: 庚 丙 丙 戊
     寅 寅 午 申

월지: 寅(인목) → 중기 병화(丙)
천간: 甲은 없고, 丙이 월간, 시간에 있음
일간: 丙(병화)
관계: 병화 vs 병화 → 비견
결과: 건록격(비견격)
```

#### 예시 3: 무토 투출 (사월) - 무토 인정 ✅
```
사주: 戊 庚 丙 戊
     巳 巳 午 申

월지: 巳(사화) → 여기 무토(戊)
천간: 丙(본기), 庚(중기) 없음, 戊만 있음
특수: 사월이므로 무토 인정 (화토동법) ✅
일간: 丙(병화)
관계: 병화 vs 무토 → 식신
결과: 식신격
```

#### 예시 4: 무토 투출 (인월) - 무토 불인정 ❌
```
사주: 戊 庚 丙 戊
     寅 寅 午 申

월지: 寅(인목) → 여기 무토(戊), 본기 갑목(甲)
천간: 甲(본기), 丙(중기) 없음, 戊만 투출
특수: 인월이므로 무토 불인정 (힘이 약함) ❌
      → 본기 갑목(甲)을 격으로 사용
일간: 丙(병화)
관계: 병화 vs 갑목 → 편인
결과: 편인격
```

#### 예시 5: 무토 투출 (해월) - 무토 불인정 ❌
```
사주: 戊 庚 丙 戊
     亥 亥 午 申

월지: 亥(해수) → 여기 무토(戊), 본기 임수(壬)
천간: 戊만 투출
특수: 해월이므로 무토 불인정 (힘이 약함) ❌
      → 본기 임수(壬)를 격으로 사용
일간: 丙(병화)
관계: 병화 vs 임수 → 편관
결과: 편관격(칠살격)
```

### 🔴 C. 진술축미(辰戌丑未) - 고지(庫地) ⭐⭐⭐ 가장 복잡!

#### 특징
- 오행의 기운이 갈무리된 창고 (Library/Storage)
- **잡기재관인격(雜氣財官印格)** 이라고 부름
- **반드시 투출해야 격이 됨** (투출 없으면 격 불완전)
- 토(土) 오행이 본기

#### 지장간 데이터 (사고지 - 四庫地)
| 월지 | 월령초기(9일) | 중기(3일) | 본기(18일) | 설명 |
|------|---------------|-----------|------------|------|
| 진(辰) | 乙(을목) | 癸(계수) | 戊(무토) | 전달(卯)의 남은 기운(여기)이 9일간 지배 |
| 미(未) | 丁(정화) | 乙(을목) | 己(기토) | 전달(午)의 남은 기운(여기)이 9일간 지배 |
| 술(戌) | 辛(신금) | 丁(정화) | 戊(무토) | 전달(酉)의 남은 기운(여기)이 9일간 지배 |
| 축(丑) | 癸(계수) | 辛(신금) | 己(기토) | 전달(子)의 남은 기운(여기)이 9일간 지배 |

⚠️ **고지(庫地) 원리**
- **초기(9일)**: 전달(前月)의 기운이 아직 강하게 남아있는 시기 - **여기(餘氣)**
- **중기(3일)**: 삼합(三合)해서 묘지로 들어가는 기운
- **본기(18일)**: 해당 계절의 토(土)가 사령하는 기간
- 총 30일: 9일 + 3일 + 18일
- 가장 변화가 많은 잡기월이므로 날짜 계산이 매우 중요

#### 우선순위 로직 (Priority Algorithm) - 자평진전 원리 적용

🚨 **핵심**: 일간(日干)이 토(土)인지 아닌지에 따라 로직이 달라집니다!

**Step 0. 지지 삼합/방합 확인 - ⭐ 최우선 순위**
```
자평진전 대원칙: "득국(得局-삼합)하면 투출한 자를 무시하고 국을 따른다"

진술축미 월인데, 지지에 삼합(三合)이나 방합(方合)이 형성되었는가?

예시:
- 진(辰)월 + 지지에 子, 申 → 신자진 수국(水局) 형성
- 축(丑)월 + 지지에 巳, 酉 → 사유축 금국(金局) 형성
- 미(未)월 + 지지에 卯, 亥 → 해묘미 목국(木局) 형성

YES:
  → 투출된 글자보다 합(合)된 오행을 격으로 잡음
  → 토(土)의 성질이 변화하여 합화된 오행이 지배함
  → 천간에 합화된 오행이 투출했다면 최강력 (성격)

NO:
  → Step 1로 이동
```

**Step 1. 본기(토) 투출 확인 & 일간 체크 - 🔥 중요!**
```
월지의 본기인 무토(戊) 또는 기토(己)가 천간에 투출?

YES:
  ├─ 일간이 토(戊/己)인가?
  │   └─ YES → 비겁(比劫)이므로 격으로 잡지 않음!
  │            → Step 2로 이동 (다른 투출 글자 찾기)
  │            → 자평진전: "비견과 겁재는 격으로 취하지 않는다"
  │
  └─ 일간이 토가 아닌가? (목/화/금/수)
      → YES → 토가 격으로 확정! (재격/관격/식상격/인격 등)
                예: 일간 갑목 + 무토 투출 → 편재격

NO:
  → Step 2로 이동
```

**Step 2. 중기/여기 투출 확인 - 잡기재관격의 핵심**
```
지장간의 중기나 여기가 투출했는가?

Case A: 하나만 투출
  → 그 글자를 격으로 잡음
  → 일간이 토여도 상관없음 (재/관/인/식이 되니까)

Case B: 둘 다 투출 (여기 + 중기 동시 투출)
  → Step 3(경합 로직) 적용

Case C: 아무것도 투출 안됨
  → Step 4로 이동
```

**Step 3. 투출 간 경합 해결 - 우선순위**
```
여기(餘氣)와 중기(中氣)가 동시에 투출했을 때:

우선순위 결정 기준:
1. 투출 수량 비교 (여기 2개 vs 중기 1개) → 수량 많은 쪽 우선
2. 사령일수 승부 (여기 9일 > 중기 3일) → 여기 우선
3. 투출 위치 (월간 > 시간 > 년간) → 월간에 있는 것 우선
4. 유정(有情) 판단 - 일간에게 더 유익한 것 (고급 로직)

일반 원칙: 여기(餘氣) > 중기(中氣)
```

**Step 4. 미투출 (아무것도 투출 안됨)**
```
창고 문이 닫혀있는 상태:

├─ 일간이 토(戊/己)가 아닌 경우:
│   → 본기(토)를 격으로 보되,
│      "격국이 아직 완성되지 않았다" 또는
│      "운에서 투출할 때 성격된다" 로 표기
│
└─ 일간이 토(戊/己)인 경우:
    → 격국 미정 (파격 또는 가색격 여부 확인 필요)
    → 충(沖-개고)이 있는지 확인 (충이 있으면 창고 문이 열림)
```

#### 로직 (Algorithm) - 자평진전 원리 반영
```javascript
function 고지_격국_판단(월지, 지지4글자, 천간4글자, 일간) {
  const 지장간 = 고지_지장간[월지]; // { 여기, 중기, 본기 }

  // Step 0: 지지 삼합/방합 확인 (최우선!)
  const 합국결과 = check삼합_방합(지지4글자, 월지);
  if (합국결과.합성립) {
    const 합화오행 = 합국결과.합화오행; // 예: 신자진 → 水
    // 합화된 오행이 천간에 투출했는가?
    const 합화투출 = 천간4글자.some(간 => get오행(간) === 합화오행);

    return {
      격용신: 합화투출 ? 합국결과.투출글자 : 합화오행,
      격명칭: calculate십성(일간, 합화오행) + "격",
      합국: true,
      합국종류: 합국결과.종류, // "신자진 수국", "해묘미 목국" 등
      투출: 합화투출,
      설명: `${월지}월이지만 ${합국결과.종류}을 이루어 ${합화오행}의 세력이 지배합니다.`
    };
  }

  // Step 1: 본기(토) 투출 확인 & 일간 체크 🔥
  if (천간4글자.includes(지장간.본기)) {
    // 일간이 토(戊/己)인가?
    const 일간오행 = get오행(일간);
    if (일간오행 === '土') {
      // 비겁은 격으로 잡지 않음 → Step 2로 넘어감
      // (아래 Step 2 로직이 처리함)
    } else {
      // 일간이 토가 아니면 → 토가 격!
      return {
        격용신: 지장간.본기,
        격명칭: calculate십성(일간, 지장간.본기) + "격",
        투출: true,
        설명: "본기 토가 투출하여 격이 됩니다."
      };
    }
  }

  // Step 2: 여기/중기 투출 수량 확인
  const 여기수량 = 천간4글자.filter(x => x === 지장간.여기).length;
  const 중기수량 = 천간4글자.filter(x => x === 지장간.중기).length;

  // Step 2-1: 여기만 투출
  if (여기수량 > 0 && 중기수량 === 0) {
    return {
      격용신: 지장간.여기,
      격명칭: calculate십성(일간, 지장간.여기) + "격",
      투출: true,
      설명: "여기가 투출하여 격이 됩니다."
    };
  }

  // Step 2-2: 중기만 투출
  if (중기수량 > 0 && 여기수량 === 0) {
    return {
      격용신: 지장간.중기,
      격명칭: calculate십성(일간, 지장간.중기) + "격",
      투출: true,
      설명: "중기가 투출하여 격이 됩니다."
    };
  }

  // Step 3: 둘 다 투출 (충돌 해결)
  if (여기수량 > 0 && 중기수량 > 0) {
    // 3-1: 수량 비교 (최우선)
    if (여기수량 > 중기수량) {
      return {
        격용신: 지장간.여기,
        격명칭: calculate십성(일간, 지장간.여기) + "격",
        투출: true,
        충돌: `여기와 중기 모두 투출, 수량상 여기 우선 (여기${여기수량}개 vs 중기${중기수량}개)`
      };
    }
    if (중기수량 > 여기수량) {
      return {
        격용신: 지장간.중기,
        격명칭: calculate십성(일간, 지장간.중기) + "격",
        투출: true,
        충돌: `여기와 중기 모두 투출, 수량상 중기 우선 (중기${중기수량}개 vs 여기${여기수량}개)`
      };
    }

    // 3-2: 수량 같음 → 사령일수 승부 (여기 9일 > 중기 3일)
    return {
      격용신: 지장간.여기,
      격명칭: calculate십성(일간, 지장간.여기) + "격",
      투출: true,
      충돌: "여기와 중기 모두 투출, 사령일수상 여기 우선 (여기9일 > 중기3일)"
    };
  }

  // Step 4: 아무것도 투출 안됨
  const 일간오행 = get오행(일간);
  if (일간오행 === '土') {
    return {
      격용신: null,
      격명칭: "격국 미정",
      투출: false,
      경고: "일간이 토이고 지장간이 투출하지 않아 격국이 불분명합니다.",
      상태: "파격 또는 가색격 여부 확인 필요"
    };
  } else {
    return {
      격용신: 지장간.본기,
      격명칭: calculate십성(일간, 지장간.본기) + "격",
      투출: false,
      경고: "지장간이 투출하지 않아 격국이 맑지 않습니다. 운에서 투출을 기다립니다.",
      상태: "격국 미완성"
    };
  }
}
```

## 🎯 내격 10가지 종류 (십성별 명칭)

격국 명칭은 **일간 vs 격용신**의 십성 관계로 결정됩니다.

**구성:**
- 정격 8가지 (재관인식 계열)
- 특수격 2가지 (건록격, 양인격)
- 월겁격 (음간의 건록격 변형, 건록격과 동급)

### 1️⃣ 정격 8가지 (재관인식 계열)

|------|-----------|------|-------------|

| 겁재 | **월겁격** | **음간(乙丁己辛癸)** + 월지가 겁재지 | 식신/재성 | 건록격과 동급, 자립심 강함 |

## 🗡️ 양인격(羊刃格) 상세 해설

### 1. 양인(羊刃)이란?

**양인(羊刃)의 의미:**
- **羊**: 양(陽), 즉 양간(陽干)의 의미
- **刃**: 칼날, 날카로운 무기
- **양인 = "양간의 칼날"** → 극도로 강하고 날카로운 기운

**본질:**
- 양인은 **일간이 왕지(旺地)에서 극도로 강해진 상태**
- 일간의 **제왕지(帝旺之)의 다음 단계**로, 기운이 정점을 넘어 날카로워짐
- 겁재(劫財) 중에서도 가장 강력한 형태
- "칼"과 같아서 잘 쓰면 천하를 얻고, 잘못 쓰면 자신을 해침

**왜 양간(陽干)만 양인이 있는가?**
- **양간(甲丙戊庚壬)**: 기운이 강하고 외향적 → 왕지에서 극도로 강해지면 "칼날"처럼 날카로워짐
- **음간(乙丁己辛癸)**: 기운이 부드럽고 내향적 → 왕지에 있어도 양인이라 부르지 않음

**음간의 월겁격(月劫格):**
- 음간 일간이 월지에서 겁재를 만나는 경우 → **월겁격**이라 부름
- 예: 을(乙) 일간 + 인(寅)월, 정(丁) 일간 + 사(巳)월
- **건록격과 동급으로 처리** (양인격처럼 특별히 취급하지 않음)
- 용신: 식신, 재성 (양인격의 칠살 아님!)
- 특징: 자립심이 강하나 양인격만큼 극단적이지 않음

### 2. 양인격 성립 조건

**필수 조건:**
```
1. 일간이 양간(甲丙戊庚壬)이어야 함 ✅ 필수!
2. 월지(月支)가 자오묘유(子午卯酉) 왕지 중 하나
3. 월지가 일간의 양인지에 해당
```

**🔥 핵심: 양인지는 모두 자오묘유(왕지)!**
| 양간 | 건록지 | 양인지 | 왕지 여부 |
|------|--------|--------|----------|
| 갑(甲) | 인(寅) | **묘(卯)** | ✅ 자오묘유 중 묘 |
| 병(丙) | 사(巳) | **오(午)** | ✅ 자오묘유 중 오 |
| 무(戊) | 사(巳) | **오(午)** | ✅ 자오묘유 중 오 |
| 경(庚) | 신(申) | **유(酉)** | ✅ 자오묘유 중 유 |
| 임(壬) | 해(亥) | **자(子)** | ✅ 자오묘유 중 자 |

**음간(乙丁己辛癸)의 경우: 양인격 없음!**
- 을(乙) + 인(寅): 월겁격 (양인격 아님, 건록격 동급으로 봄)
- 정(丁) + 사(巳): 월겁격 (양인격 아님)
- 기(己) + 사(巳): 월겁격 (양인격 아님)
- 신(辛) + 신(申): 월겁격 (양인격 아님)
- 계(癸) + 해(亥): 월겁격 (양인격 아님)

⚠️ **음간은 월겁격이라 하여 건록격과 동급으로 처리하며, 양인격처럼 특별히 취급하지 않습니다!**

### 3. 양인격의 용신 - 🔥 칠살(七殺)이 핵심!

**양인가살(羊刃駕殺) 원리:**
```
양인(칼) + 칠살(제압자) = 최강의 조합
```

**왜 칠살이 필요한가?**
1. **양인은 기운이 너무 강함** → 통제되지 않으면 폭력적, 파괴적
2. **칠살은 나를 극하는 강한 힘** → 양인을 제압할 수 있는 유일한 존재
3. **양인 + 칠살 = 강함 + 통제력** → 무예, 무력, 권력의 완벽한 조화

**양인가살의 효과:**
- 군인, 무관으로 큰 성공 → 장군, 사령관급
- 사업가로서 강력한 추진력 → 대기업 총수, CEO
- 운동선수, 격투기 선수 → 챔피언급
- 외과의사, 수술 전문의 → 정밀하고 과감한 칼 솜씨
- 정치인으로 강력한 카리스마 → 독재자형 리더

**칠살 없이 양인만 있으면?**
- 기운이 통제되지 않아 폭력적
- 형제나 동료와 갈등 (겁재의 속성)
- 재물을 모으기 어려움 (겁재가 재를 빼앗음)
- 성격이 거칠고 난폭함
- 사고나 수술, 형액(刑厄) 위험
- **파격(破格)에 가까운 상태**

### 4. 양인격의 길흉 판단

**✅ 길(吉)한 경우:**

1. **양인 + 칠살(편관)** → 양인가살격 ⭐⭐⭐⭐⭐
   - 최고의 조합
   - 무장(武將), 권력자로 대성
   - 예: 갑목 일간 + 묘월 + 천간에 경금(칠살)

2. **양인 + 정관(正官)** → 양인합관격 ⭐⭐⭐⭐
   - 칠살보다는 약하지만 길함
   - 문무를 겸비한 인재
   - 예: 갑목 일간 + 묘월 + 천간에 신금(정관)

3. **양인 + 편인(偏印)** → 양인합인격 ⭐⭐⭐
   - 학문과 힘을 겸비
   - 특수 기술, 전문직
   - 예: 갑목 일간 + 묘월 + 천간에 임수(편인)

4. **양인 + 식신(食神)** → 양인생식격 ⭐⭐⭐
   - 식신이 칠살을 제거하는 역할
   - 예술, 기술 분야에서 성공
   - 단, 관살이 있으면 식신이 관을 치므로 불길

**❌ 흉(凶)한 경우:**

1. **양인 단독** (용신 없음) → 파격 ⭐
   - 가장 위험한 경우
   - 성격이 거칠고 폭력적
   - 재물 손실, 형액, 수술, 사고 위험

2. **양인 + 양인 중복** (여러 개) → 극흉 ⭐
   - 일간, 월지, 시지, 일지에 모두 양인
   - 통제 불가능한 기운
   - 단명, 형액, 폭력 사건 위험

3. **양인 + 정재(正財) 충돌** → 흉 ⭐⭐
   - 양인(겁재)이 재성을 빼앗음
   - 부친과 불화, 처복 없음
   - 재물 손실 반복

4. **양인 + 상관** → 위험 ⭐⭐
   - 상관이 관을 극하므로 용신인 관살을 해침
   - 무법자, 반항적 성격
   - 사회 부적응

### 5. 양인의 위치별 영향

**월지(月支) 양인** → 양인격 성립 ✅
- 가장 강력한 양인
- 성격과 인생 전반에 영향
- 용신(칠살) 필수

**일지(日支) 양인** → 배우자궁에 양인
- 배우자와 갈등, 이별 가능성
- 배우자가 강한 성격
- 늦은 결혼이 유리

**시지(時支) 양인** → 자녀궁에 양인
- 자녀와 갈등 가능성
- 자녀가 강한 성격
- 만년에 파란만장

**년지(年支) 양인** → 조상궁에 양인
- 조상 덕이 없음
- 형제 중 강한 자 있음
- 어린 시절 어려움

### 7. 양인격의 대운(大運) 해석

**양인 대운 진입 시:**
- 칠살이 원국에 있으면 → 큰 성공, 권력 획득
- 칠살이 없으면 → 사고, 수술, 형액, 재물 손실

**칠살 대운 진입 시 (원국에 양인):**
- 양인가살 성립 → 최고의 운
- 승진, 사업 성공, 권력 획득
- 단, 너무 강하면 과로, 스트레스

**재성 대운 진입 시:**
- 양인(겁재)이 재물을 빼앗으려 함
- 재물 손실, 투자 실패 위험
- 여자 문제, 부친 문제

**식상 대운 진입 시:**
- 식신이 칠살을 제거 (설기제살)
- 관살이 용신인데 식신이 치면 흉
- 상관이면 더욱 흉 (상관견관)

### 9. 양인격 판단 로직 (프로그래밍)

```javascript
function check양인격(일간, 월지, 천간4글자) {
  // Step 1: 양간인가?
  const 양간 = ['甲', '丙', '戊', '庚', '壬'];
  if (!양간.includes(일간)) {
    return { 양인격: false, 이유: "음간은 양인격 없음" };
  }

  // Step 2: 월지가 양인지인가?
  const 양인지 = {
    '甲': '卯', '丙': '午', '戊': '午', '庚': '酉', '壬': '子'
  };

  if (월지 !== 양인지[일간]) {
    return { 양인격: false, 이유: "월지가 양인지 아님" };
  }

  // Step 3: 용신(칠살) 확인
  const 칠살천간 = {
    '甲': '庚', '丙': '壬', '戊': '甲', '庚': '丙', '壬': '戊'
  };

  const 칠살있음 = 천간4글자.includes(칠살천간[일간]);

  if (칠살있음) {
    return {
      양인격: true,
      격명칭: "양인가살격",
      용신: 칠살천간[일간],
      평가: "최고의 격국, 대성할 운명",
      적성: "무관, 군인, 사업가, 외과의사"
    };
  }

  // Step 4: 다른 용신 확인 (정관, 편인 등)
  const 정관천간 = {
    '甲': '辛', '丙': '癸', '戊': '乙', '庚': '丁', '壬': '己'
  };

  if (천간4글자.includes(정관천간[일간])) {
    return {
      양인격: true,
      격명칭: "양인합관격",
      용신: 정관천간[일간],
      평가: "길격, 문무겸비",
      적성: "공무원, 관리직, 전문직"
    };
  }

  // Step 5: 용신 없음 (파격)
  return {
    양인격: true,
    격명칭: "양인격(파격)",
    용신: null,
    평가: "기운 과다, 통제 필요",
    경고: "칠살을 만나야 안정됨",
    위험: "사고, 수술, 형액, 재물손실"
  };
}
```

### 10. 양인격 핵심 요약

**✅ 반드시 기억할 것:**
1. 양인 = **오직 양간(甲丙戊庚壬)만 해당!** (음간은 양인격 없음)
2. 양인지는 모두 **자오묘유(왕지)** - 갑묘, 병오, 무오, 경유, 임자
3. 월지가 양인지일 때만 양인격 성립
4. **칠살(편관)이 용신** - 없으면 파격
5. 양인가살 = 최강의 조합 (무장, 권력자)
6. 칠살 없으면 = 위험 (통제 불가)
7. 직업: 무관, 의사, 사업가, 운동선수
8. 배우자궁에 양인 = 결혼 문제
9. **음간(乙丁己辛癸)은 월겁격** - 건록격과 동급으로 처리

**⚠️ 주의사항:**
- 양인은 양날의 검 → 잘 쓰면 천하, 못 쓰면 재앙
- 대운에서 양인 만나면 사고, 수술 조심
- 재성 대운은 재물 손실 위험
- 반드시 칠살을 찾아야 함!
- **음간은 양인격이 아니라 월겁격임을 명심!**

#### 2. 🟢 사생지(四生地): 인신사해 (寅申巳亥) ✅ 완료
```
월령초기(7일) + 중기(7일) + 본기(16일) = 30일

- 인(寅): 무(7) - 병(7) - 갑(16)
- 신(申): 무(7) - 임(7) - 경(16)
- 사(巳): 무(7) - 경(7) - 병(16)
- 해(亥): 무(7) - 갑(7) - 임(16)

특징: 여기(餘氣)는 모두 무토(戊), 전달의 남은 기운
정설: 7-7-16 사용 (대부분의 만세력과 이론서)
```

#### 3. 🟠 사고지(四庫地): 진술축미 (辰戌丑未) ✅ 완료

```
   - **양인격**: 양간(甲丙戊庚壬) + 자오묘유(왕지) → 용신 칠살!
```
