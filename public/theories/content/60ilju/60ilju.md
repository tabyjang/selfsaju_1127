# 60일주(六十日柱) 완전 가이드

![60일주](/60ilju/60ilju-main.webp)

60일주는 사주명리학에서 가장 중요한 요소 중 하나입니다. **일주(日柱)**는 태어난 날의 천간과 지지를 조합한 것으로, 그 사람의 **본질적인 성격과 운명**을 나타냅니다.

---

## 60갑자란?

**60갑자(六十甲子)**는 10개의 천간(天干)과 12개의 지지(地支)가 순환하며 만들어내는 60가지 조합입니다.

### 10천간(天干)

| 천간 | 한자 | 오행 | 음양 |
|------|------|------|------|
| 갑 | 甲 | 목(木) | 양(陽) |
| 을 | 乙 | 목(木) | 음(陰) |
| 병 | 丙 | 화(火) | 양(陽) |
| 정 | 丁 | 화(火) | 음(陰) |
| 무 | 戊 | 토(土) | 양(陽) |
| 기 | 己 | 토(土) | 음(陰) |
| 경 | 庚 | 금(金) | 양(陽) |
| 신 | 辛 | 금(金) | 음(陰) |
| 임 | 壬 | 수(水) | 양(陽) |
| 계 | 癸 | 수(水) | 음(陰) |

### 12지지(地支)

| 지지 | 한자 | 오행 | 띠 |
|------|------|------|-----|
| 자 | 子 | 수(水) | 쥐 |
| 축 | 丑 | 토(土) | 소 |
| 인 | 寅 | 목(木) | 호랑이 |
| 묘 | 卯 | 목(木) | 토끼 |
| 진 | 辰 | 토(土) | 용 |
| 사 | 巳 | 화(火) | 뱀 |
| 오 | 午 | 화(火) | 말 |

| 유 | 酉 | 금(金) | 닭 |

## 일주 분석의 핵심 요소

### 1. 십이운성(十二運星)

일주에서 일간이 일지에서 어떤 에너지 상태인지를 나타냅니다.

| 운성 | 의미 | 특징 |
|------|------|------|
| 장생(長生) | 탄생 | 새로운 시작, 성장 |
| 목욕(沐浴) | 성장 | 변화, 불안정 |
| 관대(冠帶) | 청년 | 의욕, 패기 |
| 건록(建祿) | 장년 | 안정, 실력 |
| 제왕(帝旺) | 정점 | 최고조, 자신감 |
| 쇠(衰) | 쇠퇴 | 안정, 원숙 |
| 병(病) | 병듦 | 예민, 섬세 |
| 사(死) | 정지 | 침착, 냉정 |
| 묘(墓) | 저장 | 축적, 내면 |
| 절(絶) | 단절 | 새로움, 변화 |
| 태(胎) | 잉태 | 가능성, 준비 |
| 양(養) | 양육 | 보호, 성장 |

### 2. 일지 십성(十星)

일지에 있는 십성이 그 사람의 배우자궁과 일상을 나타냅니다.

| 십성 | 의미 | 일지에서의 특징 |
|------|------|-----------------|
| 비견 | 형제 | 독립심, 경쟁심 |
| 겁재 | 경쟁자 | 승부욕, 손재수 |
| 식신 | 표현 | 먹을 복, 재능 |
| 상관 | 창의 | 예술성, 비판정신 |
| 편재 | 활동재 | 투자, 활동성 |
| 정재 | 안정재 | 저축, 성실함 |
| 편관 | 권력 | 추진력, 압박감 |
| 정관 | 명예 | 책임감, 안정 |
| 편인 | 직감 | 예술성, 독창성 |
| 정인 | 학문 | 지혜, 보호 |

### 3. 특수 신살(神殺)

일주에 붙는 특별한 신살들입니다.

| 신살 | 의미 |
|------|------|

| 역마살 | 이동, 변화 |
| 도화살 | 매력, 인기 |
| 양인살 | 강한 에너지 |

## 60일주 활용법

1. **자신의 일주 확인**: 만세력으로 자신의 사주를 확인하세요
2. **일주 특성 파악**: 해당 일주의 상세 페이지에서 성격과 특성을 확인하세요
3. **강점과 약점 이해**: 타고난 강점을 살리고 약점을 보완하세요
4. **궁합 확인**: 배우자나 파트너와의 궁합을 확인해 보세요
5. **직업 적성**: 자신에게 맞는 직업 분야를 탐색하세요
