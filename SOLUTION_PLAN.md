# 일주 특성 다양화 해결 방안

## 문제점
- 일주 특성을 열심히 추가했는데 제대로 활용 안 됨
- 반복되거나, 아예 안 쓰거나
- 다양성 부족

## 해결 방안

### 1. 제목 (Title)
**현재**: 템플릿에서 고정된 특성 사용
**개선**: 랜덤하게 다양한 특성 선택
- 핵심특성, 강점, 소통스타일, 감정표현, 업무스타일 중 **랜덤 선택**

### 2. 오프닝 (Opening)
**현재**: 핵심특성 + 다른 특성 (2개 중복)
**개선**: **하나의 특성만** 랜덤 선택
```
5가지 옵션:
1. 핵심특성만: "차분하고 신중한 당신, 오늘을 보내세요"
2. 강점만: "책임감 있는 당신, 오늘을 보내세요"
3. 소통스타일만: "말을 아끼고 행동으로 보여주는 당신, 오늘을..."
4. 감정표현만: "감정을 내면에 담아두는 당신, 오늘을..."
5. 업무스타일만: "묵묵히 일하며 성과를 내는 당신, 오늘을..."
```

### 3. 메인 이벤트
**현재**: 일주 특성 없음
**개선**: 일주 특성 플레이스홀더 추가
```
"재미": [
  "유쾌한 하루! {ilju.핵심특성} 당신도 웃음 지을 거예요",
  "오후 2-3시쯤 {ilju.소통스타일} 당신의 스타일과 딱 맞는..."
]
```

### 4. 에너지 조합
**현재**: 일주 특성 없음
**개선**: 일주 특성 플레이스홀더 추가
```
"활동낮음_마음낮음": "충전이 필요해요. {ilju.핵심특성} 당신답게 푹 쉬고..."
```

### 5. 요일별 테마
**현재**: 일주 특성 없음
**개선**: 일주 특성 플레이스홀더 추가
```
"월요일": "새로운 한 주! {ilju.핵심특성} 당신답게 시작하면..."
```

## 구현 순서
1. ✅ 오프닝 단순화 (하나의 특성만 사용)
2. ✅ JSON에 플레이스홀더 추가
3. ✅ dataLoader에서 다양한 seed 사용
4. ✅ 공백 문제 수정
